# AGENTS.md

When working on this website, always follow [`COPY_RULES.md`](./COPY_RULES.md).

Non-negotiable copy rule:
- Always use the site tone of voice: clear before clever, direct, helpful, dry, and lightly cheeky.

Product clarity rule:
- Always say plainly that Explain It Like I’m Five is a skill for AI agents.
- Make clear that it can be used in tools like Codex, Claude Code, Cursor, and similar agents.
- Lead with what the skill does before leaning on style, humor, or metaphor.

Hero rule:
- The hero should explain the product in one glance.
- A visitor should not need to scroll to learn what the skill is.

Editing rule:
- If a line sounds polished but could belong to any startup landing page, cut it or rewrite it.
